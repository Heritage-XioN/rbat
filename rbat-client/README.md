# rbat-client

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg?style=for-the-badge)](https://opensource.org/licenses/Apache-2.0)
[![Build Status](https://img.shields.io/github/actions/workflow/status/Heritage-XioN/rbat/ci-client.yml?style=for-the-badge&label=build)](https://github.com/Heritage-XioN/rbat/actions)
[![Next.js Version](https://img.shields.io/badge/Next.js-15.0+-black.svg?style=for-the-badge&logo=nextdotjs)](https://nextjs.org/)

**rbat-client** is the Next.js-based terminal-native web interface for the Rust Binary Analysis Tool (RBAT). It allows users to drag-and-drop binaries for inspection, forwards uploads to the gRPC backend daemon, and displays real-time analysis logs, section-level entropy heatmaps, and security vulnerability scores in a dashboard.

---

## Features

* **Cyberpunk Cyber-Security UI**: A premium dark-mode interface designed with rich terminal aesthetics, interactive section navigation, and visual cues.
* **gRPC Streaming upload Integration**: Integrates directly with the `rbat-server` daemon, dividing and streaming uploaded binaries to object storage via gRPC.
* **Real-Time SSE Updates**: Listens to server-sent events (SSE) generated by backend webhook notifications, allowing pages to display progress updates dynamically.
* **AI Analysis Insights**: Stream-generates structural summary analysis reports and actionable security recommendations using the Gemini 2.5 Flash API.
* **Encrypted Webhook Verification**: Implements request signature authentication using `standardwebhooks` to ensure secure communications between client and server.

---

## Configuration

The client requires a set of environment variables to run. Create a `.env` file in the root of the client directory:

| Variable | Description | Default |
|---|---|---|
| `WEBHOOK_SECRET` | Secret key used to verify incoming webhook signatures from `rbat-server` | |
| `GRPC_SERVER_URL` | The host address and port of the gRPC server | `localhost:8080` |
| `WEBHOOK_RECEIVER_URL` | The callback URL where `rbat-server` sends webhook updates | `http://localhost:8080/webhooks` |
| `GEMINI_API_KEY` | Your Google Gemini API key for generating AI analysis summaries | |

## Quick Start

### Installation

Install dependencies using `pnpm` (recommended) or `npm`:

```bash
pnpm install
```

### Running the Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Building for Production

Compile a production-ready optimized build:

```bash
pnpm build
pnpm start
```
